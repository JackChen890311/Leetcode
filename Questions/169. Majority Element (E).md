 - 四個字：諸侯對拼 XD
 ```python
 class Solution:
    def majorityElement(self, nums: List[int]) -> int:
        maj, freq = nums[0], 1
        for n in nums[1:]:
            if n != maj and freq > 0:
                freq -= 1
            elif n != maj and freq == 0:
                maj = n
            elif n == maj:
                freq += 1
        return maj
 ```
