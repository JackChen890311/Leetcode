```python
    def firstBadVersion(self, n: int) -> int:
        left, right = 0, n
        while left <= right:
            mid = (left + right) // 2
            if not isBadVersion(mid):
                left = mid + 1
            else:
                right = mid - 1
        return left
```
- 終止狀況
	- [F, F, T, T] -> target: 2
		- L = R = 1 -> L = mid + 1 = 2
		- L = R = 2 -> R = mid - 1, L = 2
		- L = 1, R = 2 -> mid = 1 -> L = mid + 1 = 2
		- L = 2, R = 3 -> mid = 2 -> R = mid - 1 = 1
		- L = 0, R = 2 -> mid = 1 -> L = mid + 1 = 2
		- L = 1, R = 3 -> mid = 2 -> R = mid - 1 = 1 -> L = R = 1
		- L = 0, R = 3 -> mid = 1 -> L = mid + 1 = 2
